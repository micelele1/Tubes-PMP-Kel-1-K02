# Tubes-PMP-Kel-1-K02
Tugas Besar EL2008 Pemecahan Masalah dengan Pemograman
Setiap data inventaris setidaknya memuat: ID barang, nama komponen/peralatan, kategori barang, jumlah stok, lokasi penyimpanan, status barang (misalnya tersedia, dipinjam, rusak, atau habis), pemilik barang serta PIC untuk barang tersebut. 
 Database harus diimplementasikan pada Arduino Uno, sementara operasi dasar inventaris yang dapat dilakukan melalui Serial Monitor meliputi menambah data, menghapus data, mencari data berdasarkan ID barang, memperbarui stok, memperbarui status dan menampilkan seluruh data, dan menampilkan ringkasan inventaris. 
Program wajib memenuhi ketentuan: main hanya boleh digunakan sebagai pemanggil fungsi serta seluruh fungsi selain main harus bertipe data void; 
Program harus menangani berbagai permasalahan: ID duplikat, ID tidak ditemukan, stok tidak mencukupi, data kosong, kapasitas memori hampir habis, penghapusan node pertama, tengah, dan akhir linked list.
